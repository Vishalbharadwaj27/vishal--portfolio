## 📌 Overview

This portfolio application presents:

- Featured projects with structured descriptions
- Technical skills categorized by stack
- Professional summary
- Contact information
- Responsive and optimized UI

The application is built with a modular component structure and deterministic rendering behavior.

## 🛠 Tech Stack

Frontend:
- React
- TypeScript
- Vite
- Modern CSS / Utility-based styling

Deployment:
- Vercel

- Git + GitHub

## 📂 Project Structure

src/
components/
pages/
assets/
styles/
main.tsx
index.html
vite.config.ts


- `components/` – Reusable UI modules
- `pages/` – Route-level views
- `assets/` – Static assets
- `main.tsx` – Application entry point
- `vite.config.ts` – Build configuration
